# 📞 SnickerShow - Call Me If You Get Lost Travel License Generator

A professional, multi-page website featuring a free vintage Travel License generator inspired by Tyler, The Creator's "Call Me If You Get Lost" album.

## 🎯 Features

- **Free Travel License Generator** - Vintage passport-style design, photo upload, PNG/PDF download
- **Multi-Page Website** - Home, About, Contact, Privacy Policy, Disclaimer
- **SEO Optimized** - Meta tags, Open Graph, Schema markup
- **Mobile Responsive** - Works perfectly on all devices
- **No Login Required** - Privacy-focused, no data collection
- **Professional Design** - Clean header/footer navigation

## 🎨 Travel License Generator Features

### **Vintage Design:**
- Authentic old passport/license aesthetic
- Star border pattern
- Typewriter font (Special Elite)
- Aged paper texture
- Dotted line fields
- "CALL ME IF YOU GET LOST" stamp

### **Input Fields:**
- Issued To (Full Name)
- License Number (e.g., TTC6252021)
- Date of Birth
- Place of Issue
- Date of Issue
- Photo Upload
- Signature (cursive font)

### **Functionality:**
- Real-time preview
- Photo upload with validation
- PNG download (high quality)
- PDF download (license size)
- Randomize button
- Auto-save to localStorage
- Keyboard shortcuts (Ctrl+S, Ctrl+P, Ctrl+R)
